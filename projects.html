Projects

Here are described all the most important projects where i was involved. Unfortunately not all of them are open-source, so it's not possible to have a look at the code for everyone. Nevertheless i explain for all of them what is/was my role in summary within the development.

Current projects

Politecnico di Torino

The name of our project is ETS, standing for ESP tracking system. The project is under development for the course "Programmazione di sistema". The goal of the project is the triangulation and analysis of mobile devices in buildings. The RSSI for each packed are measured through ESP32 Devices, placed around the rooms. Data are sent to a server that elaborate analysis results and store them in a MySQL database. The results can be visualized through a GUI application. The project will be presented in upcoming month and source code will be public for that date.

EURECOM
I'm working on a university project related to asynchronous machine learning. The scope is to analyze how the staleness of models influence the convergence. I'm working with two other students, and now we are exploring the state of the art of the subject.

Past Projects

I did some contributions to a computational electromagnetics framework. I implemented WIRE EFIE formulation and some other features within the framework. The job required knowledge about physics, linear algebra, maths and algorithms.

Politecnico di Torino

This project was finalized to academic purposes. It is concerned to a specific exam about computer architectures, where we got the opportunity to work with Raspberry pi 3. The goal of our project is to determine the delay introduced by the Operative System of the Raspberry pi 3 exploiting different methods and libraries. The work of test has been performed in the laboratory using some electronic instruments such as the oscilloscope and the waveform generator. The source code is available on my github profile.
