Politecnico di Torino

ETS: Esp32 Tracking System
The name of our project is ETS, standing for ESP tracking system. The project was developed for the course System Programming. The goal of the project is the triangulation and analysis of mobile devices in buildings. The RSSI for each packed are measured through ESP32 Devices, placed around the rooms. Data are sent to a server that elaborate analysis results and store them in a MySQL database. The results can be visualized through a GUI application. The source code is available on my GitHub account.
